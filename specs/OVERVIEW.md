# etraproject.ru — B2C продающий сайт

## Стек

| Слой | Технология |
|------|-----------|
| Framework | Next.js 15 (App Router) |
| CMS / Backend | Payload CMS 3.x (встроен в Next.js) |
| БД | PostgreSQL 16 (Timeweb DBaaS) |
| ORM | Drizzle (Payload PostgreSQL adapter) |
| Файлы / Медиа | Timeweb S3 (`@payloadcms/storage-s3`) |
| Стили | Tailwind CSS 4 |
| UI | shadcn/ui + Radix |
| Иконки | Lucide |
| Анимации | Framer Motion |
| Формы | React Hook Form + Zod |
| Оплата | ЮKassa / Тинькофф Эквайринг |
| Email | Resend / SMTP (Timeweb корп. почта) |
| Аналитика | Яндекс.Метрика, GA4 |
| CRM-интеграция | amoCRM REST API |
| Деплой | Docker → Timeweb App Platform |

## Публичная часть (Frontend)

## Админка (Payload CMS)

### 1. CMS (контент)
- **Страницы** — лендинг, о проекте, контакты (блоковый редактор)
- **Статьи / Блог** — Rich Text, категории, теги, SEO-поля, публикация/черновик
- **Рецепты** — ингредиенты, шаги, сложность
- **FAQ** — вопрос-ответ, порядок
- **Баннеры / Слайдеры** — hero, промо
- **Отзывы** — модерация, рейтинг
- **Навигация** — меню, footer links

### 2. Каталог
- **Товары** — название, slug, описание, цена, старая цена, SKU, галерея, варианты, категория, статус (активен/скрыт), SEO
- **Категории** — дерево, иконка, slug, порядок
- **Промокоды** — код, скидка (% / фикс.), срок, лимит использований

### 3. Заказы
- Список заказов с фильтрами (статус, дата, сумма)
- Детализация заказа (товары, клиент, адрес, оплата)
- Смена статуса (новый → обработка → отправлен → доставлен → завершён)
- Экспорт в CSV
- Синхронизация с amoCRM (сделки)

### 4. Клиенты
- Список клиентов
- Карточка клиента (профиль, заказы, сумма покупок)
- Роль (клиент / VIP / заблокирован)
- Синхронизация с amoCRM (контакты)

### 5. Медиа
- Библиотека медиафайлов (S3)
- Загрузка изображений, видео, документов
- Автоматический ресайз / WebP-конвертация
- Папки / теги

### 6. SEO
- Глобальные мета-теги (title template, OG image)
- robots.txt, sitemap.xml (автогенерация)
- Редиректы (301/302)
- JSON-LD разметка (Organization, Product, Article, BreadcrumbList)

### 7. Настройки
- **Общие** — название сайта, логотип, favicon, контакты
- **Интеграции** — ключи amoCRM, ЮKassa, SMTP, S3
- **Доставка** — способы, тарифы, регионы
- **Уведомления** — шаблоны email (заказ, регистрация, восстановление)

## Интеграции

### amoCRM
- **Контакты** — при регистрации / заказе → создание/обновление контакта
- **Сделки** — при оформлении заказа → создание сделки в воронке
- **Каталог** — синхронизация товаров (amoCRM = источник истины)
- **Webhook'и** — двусторонняя синхронизация статусов

### Оплата (ЮKassa / Тинькофф)
- Создание платежа → редирект → callback
- Подтверждение / отмена / возврат
- Фискализация (54-ФЗ)

### Доставка
- CDEK API — расчёт стоимости, создание заявки, трекинг
- Почта России — альтернативный канал
- Самовывоз — точки выдачи

### S3 (Timeweb)
- Все медиа через `@payloadcms/storage-s3`
- CDN для статики
- Автоматический WebP

### Email
- Transactional: подтверждение заказа, регистрация, восстановление пароля
- Шаблоны в Payload (Globals)

## SEO и производительность

- **SSG / ISR** для каталога, статей, лендинга (revalidate 60-300s)
- **SSR** для ЛК, корзины, checkout
- **Edge Middleware** для геолокации / A-B тестов
- **Image Optimization** — Next.js `<Image>` + S3 + sharp
- **Core Web Vitals** — target LCP < 2.5s, CLS < 0.1
- **sitemap.xml** — автогенерация из коллекций Payload
- **robots.txt** — управление из админки
- **Structured Data** — JSON-LD для всех типов страниц
